系統資訊

遊戲名稱 Wall Go (牆壁戰棋)
目前版本 v3.0 終極版
核心技術 Vanilla JS + CSS Grid
開發者 Kuoli

結合了 BFS 領地掃描與 L 型碰撞偵測機制的原創網格策略桌遊。

📜 遊戲規則

• 放置期：依序放完 2 顆棋子。
• 行動期：每回可移動 1~2 步（含 L型轉彎，不可越子）。
• 建牆：移動後必須在該棋子四周蓋牆。
• 破牆：每人一次，可打破任何一道牆。
• 限時：每回合 90 秒，超時將剝奪移動權並隨機建牆。
• 獲勝：佔領最多「純淨獨占領地」者勝。
